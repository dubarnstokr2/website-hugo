Victor Hugo

A Hugo boilerplate for creating truly epic websites

This is a boilerplate for using Hugo as a static site generator and Webpack 5 as your asset pipeline. Victor Hugo setup to use PostCSS v8 and Babel v7 for CSS and JavaScript compiling/transpiling.
